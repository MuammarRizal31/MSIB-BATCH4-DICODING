Pengembang Cloud Profesional dapat membuat aplikasi yang scalable menggunakan praktik dan alat yang direkomendasikan oleh berbagai Cloud Solution Provider. Seorang Cloud Developer dapat sepenuhnya memanfaatkan layanan yang disediakan oleh Cloud Solution Provider tersebut. Individu ini memiliki pengalaman dengan basis data terkini (next-gen database), lingkungan di mana aplikasi berjalan (runtime environment), dan alat pengembang (developer tools). Mereka juga memiliki kemampuan dengan setidaknya satu bahasa pemrograman umum dan terampil menghasilkan metrik dan log yang bermakna untuk debug dan melacak kode.

1. HEMAT BIAYA
    - Sebab,cloud umumnya menggunakan model pembayaran pay-as-you-go atau bayar sesuai apa yang kita gunakan.
    
2. Skalabilitas
    - Dengan cloud, Anda bisa menambah atau mengurangi sumber daya yang digunakan sesuai dengan kebutuhan dengan cepat dan mudah.
    
3. elastis
    - Beban kerja yang diterima oleh server atau website bisa sangat fluktuatif tergantung dengan kondisi perusahaan.

4. Bisa diandalkan
    - Cloud mempunyai fitur backup dan replikasi data. Tujuannya untuk menjaga data tetap aman meskipun terjadi hal seperti kegagalan sistem ataupun bencana alam yang meluluhlantakkan data center.
    
5. Global
    - Cloud provider biasanya tidak hanya memiliki satu data center saja, namun ada banyak data center yang tersebar di beberapa wilayah di dunia. 

6. Aman 
    - Cloud provider menyediakan sejumlah kebijakan, teknologi, dan kontrol bagi Anda untuk melindungi data, aplikasi, dan infrastruktur yang Anda kembangkan di cloud dari berbagai potensi ancaman.

Kebutuhan penting nomer wahid yaitu hardware yang meliputi komputer. Berikut spesifikasi dari hardware yang diperlukan untuk berkarier sebagai Azure Cloud Developer:

    1. Microsoft Windows, Linux, atau MacOS. 
    2. Prosesor Intel Dual Core (Rekomendasi Core i3 ke atas).

    4. Resolusi layar 1280x800 (Rekomendasi Full HD 1920 x 1080).

Spesifikasi yang dibutuhkan untuk membuat aplikasi di cloud tidak terlalu tinggi. Tentu semakin tinggi spesifikasi, akan membantu Anda lebih cepat dalam membangun aplikasi tersebut.

> SOFTWARE
Yang Anda butuhkan teks editor. Beberapa sistem operasi sebenarnya sudah menyediakan text editor usungannya sendiri. Contohnya Windows memiliki Notepad, Linux memiliki Text Editor, dan Macintosh memiliki TextEdit. Ketiga aplikasi tersebut bisa kita gunakan untuk belajar menulis kode.

Selanjutnya, untuk mengakses layanan cloud membutuhkan sebuah browser. Anda bisa menggunakan browser apa pun untuk mengunjungi website tersebut. Browser juga dapat membantu Anda dalam melakukan banyak hal, mencari dokumentasi, bertanya di forum-forum, mencari jawaban dari permasalahan yang Anda alami, dan lain-lain. 

Selain itu Anda dapat menginstall docker untuk mengelola aplikasi web dengan mudah. Docker adalah platform perangkat lunak yang memungkinkan Anda membuat, menguji, dan menerapkan aplikasi dengan cepat menggunakan kontainer.

Sedangkan container/kontainer adalah sebuah mekanisme pengemasan aplikasi yang dapat diabstraksi dari lingkungan aplikasi tersebut dijalankan. Singkatnya, container adalah paket aplikasi yang memungkinkan kita men-deploy dengan cepat dan konsisten di beberapa lingkungan yang berbeda seperti data center, cloud publik, ataupun perangkat personal dari developer.

> SKILL DAN PENGETAHUAN
Lalu apa saja yang perlu Anda pelajari? Tentu yang Anda perlu pelajari pertama adalah bahasanya. Untuk cloud computing, mempunyai berbagai bahasa yang bisa Anda gunakan, seperti Java, C#, JS, Golang, Ruby, Python dan lain-lain. Bahasa tersebut sesuai dengan layanan yang Anda pilih.

Selain itu, ada beberapa poin yang perlu Anda pelajari juga untuk menjadi Azure Cloud Developer, yakni:

    * Dasar Arsitektur dan Layanan Azure
        - Dapat membangun infrastruktur dengan menjalankan teknologi cloud dari Azure, serta dapat menggunakan berbagai jenis layanan Azure dengan baik.
    
    * Dasar Komputasi
        - Dapat menggunakan layanan komputasi Azure, seperti Virtual Machines (VM), Containers, dan Serverless.
        
    * Dasar Penyimpanan dan Database
        - Dapat menggunakan dengan baik jenis-jenis layanan penyimpanan untuk menyimpan dan mengelola data Anda di cloud.
        
    * Dasar Jaringan
        - Mempelajari bagaimana layanan Azure saling berkomunikasi dan bagaimana membuat aplikasi Anda selalu tersedia bagi pengguna.
        
    * Dasar Keamanan
        - Dapat melindungi data dan aplikasi Anda dari serangan.

    * Monitor, Troubleshoot, and Optimize Solutions
        - Dapat memantau dan mengoptimasi solusi aplikasi dengan Azure.
    
    * Azure and Third-party Services
        - Dapat mengintegrasikan layanan Azure dengan layanan pihak ketiga.
