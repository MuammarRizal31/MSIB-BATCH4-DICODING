> HARDWARE
kebutuhan penting nomer wahid yaitu hardware yang meliputi komputer.Berikut spesifikasi dari hardware yang diperlukan untuk berkarier sebagai Machine Learning Developer:

    1. Microsoft Windows, Linux, atau MacOS.
    2. Prosesor Intel Dual Core (Rekomendasi Core i3 ke atas).
    3. RAM sebesar minimum 2 GB (Rekomendasi 4 GB).
    4. Terdapat 2 GB ruang kosong pada penyimpanan (Rekomendasi 4 GB).
    5. Resolusi layar 1280x800 (Rekomendasi Full HD 1920 x 1080).
    
Spesifikasi yang dibutuhkan tidak terlalu tinggi untuk membuat sebuah model dalam machine learning. Tentu semakin tinggi spesifikasi, akan membantu Anda lebih cepat dalam membangun aplikasi tersebut.

> SOFTWARE
Yang Anda butuhkan pertama kali adalah IDE. Integrated Development Environment (IDE) lebih dari sekedar text editor untuk membuat kode, di dalamnya tergabung juga berbagai fasilitas development misalnya Code Versioning, Interpreter, Visualization, dan lain sebagainya.

Untuk menulis program pada bahasa Python, bahasa populer untuk menjalankan machine learning, Anda dapat menggunakan teks editor apa pun, misalnya Notepad++ atau sejenisnya (Windows), Nano, Vim, Gedit (Linux), atau Visual Studio Code/Atom/Sublime (semua platform). Anda juga bisa menggunakan IDLE dalam bundel package python atau IDE lain yang khusus dibuat untuk Python, misal PyCharm dari JetBrains.

Selain itu, Anda juga dapat memanfaatkan IDE lain misalnya Jupyter Notebook. Jupyter Notebook adalah aplikasi web open-source yang memungkinkan Anda membuat dan berbagi dokumen yang berisi kode langsung, persamaan, visualisasi, dan teks naratif. Kegunaannya meliputi: pembersihan dan transformasi data; simulasi numerik; pemodelan statistik; visualisasi data; pembelajaran mesin; dan masih banyak lagi.

IDE yang paling saya rekomendasikan adalah Google Colaboratory atau yang bisa disebut Google Colab. IDE yang dapat digunakan untuk menulis dan mengeksekusi Python di browser Anda dengan beberapa fitur seperti tidak perlu konfigurasi, akses gratis ke GPU, dan berbagi dengan mudah. Sehingga dapat mempermudah pekerjaan Anda.

Dengan Google Colab, Anda dapat mengimpor kumpulan data gambar, melatih klasifikasi gambar di atasnya, dan mengevaluasi model. Semua hanya dilakukan dalam beberapa baris kode. Notebook Colab mengeksekusi kode di server Google Cloud, yang berarti Anda dapat memanfaatkan kekuatan hardware Google dan yang Anda butuhkan hanyalah browser.
